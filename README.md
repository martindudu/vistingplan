# 🗺️ 旅遊建築師 Pro (Travel Architect Pro)

一個極致現代化的旅遊行程規劃工具。結合了 Google Maps 的強大功能與奢華的磨砂玻璃設計感，提供全方位的旅程規劃體驗。

## ✨ 頂級功能特色

### 🎨 職人級視覺設計
- **沉浸式 UI**：動態背景圖與磨砂玻璃 (Glassmorphism) 卡片設計。
- **持久地圖模式**：地圖固定於側邊，規劃時路徑變化一目了然。
- **暗夜銀地圖**：深度客製化的地圖樣式，減少長途規劃的視覺疲勞。

### 🕒 精確時間軸管理
- **每日啟程設定**：設定出發時間，系統自動推算整日時刻表。
- **停留時間自訂**：為每個景點設定預計停留時長。
- **多天行程規劃**：支援無限天數分頁，靈活管理長途旅行。

### 🚀 智慧規劃工具
- **智慧路線最佳化**：一鍵重新排序景點，尋找行車時間最短的路徑。
- **周邊探索模式**：自動推薦景點、美食與住宿，一鍵點擊即可加入行程。
- **實境街景搶先看**：直接在地圖中預覽景點實景。
- **即時天氣預報**：顯示每個景點的當前氣溫與氣候。

### 📲 分享與行動支援
- **互動分享連結**：將完整行程加密打包成網址，旅伴開啟即可完全復原。
- **精美海報匯出**：一鍵生成高品質的直式長圖海報，方便分享至 LINE 或 IG。
- **PWA 支援**：可將網頁安裝至手機桌面，體驗如原生 App 般的流暢。

## 🚀 快速開始

### 本地開發
1. **安裝依賴**：`npm install`
2. **設定環境變數**：建立 `.env.local` 並填入 `NEXT_PUBLIC_GOOGLE_MAPS_API_KEY`；若要啟用 AI 規劃，另外加入 `OPENAI_API_KEY`。
3. **啟動**：`npm run dev`

### 部署到 Vercel
1. 將專案推送到您的 GitHub 倉庫。
2. 在 Vercel 導入此專案。
3. 務必在 Vercel 專案設定中加入您的 API Key。

## 📝 使用技巧
- **分享行程**：點擊右上角的「分享」按鈕，系統會複製文字摘要與一個互動連結。
- **街景探路**：點擊卡片上的「👁️」眼睛圖示，地圖會切換為 360 度實景。
- **調整順序**：直接按住卡片左側拖曳，時間軸會自動重新計算。

## 🔒 安全注意事項
- 本專案不儲存使用者隱私，所有行程均儲存於您的瀏覽器 (Local Storage) 或網址參數中。
- 請務必為您的 Google Maps API Key 設定網域限制。
